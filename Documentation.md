# EXPENSE TRACKER USING PYTHON

## 1. Abstract

The Expense Tracker is a Python-based application designed to help users manage their daily expenses efficiently. The system allows users to add, view, and delete expense records while calculating the total amount spent. Expense data is stored in a JSON file, ensuring permanent storage and easy retrieval. This project demonstrates the practical implementation of Python programming concepts such as functions, file handling, lists, dictionaries, loops, and exception handling.

## 2. Introduction

Managing daily expenses is an important aspect of personal finance. Keeping track of expenditures helps users understand their spending habits and manage their budget effectively. The Expense Tracker application provides a simple and user-friendly solution for recording and monitoring expenses. The application stores expense details such as amount, category, and date, allowing users to maintain a record of their financial activities.

## 5. Methodology

The application follows a menu-driven approach. Users can choose different options such as adding, viewing, deleting expenses, or calculating total spending. Expense records are stored in a JSON file. When the program starts, it loads existing data from the file. Whenever a user performs an operation, the updated data is saved back to the JSON file.

## 6. Algorithm

1. Start the program.
2. Load expense records from the JSON file.
3. Display the main menu.
4. Accept user choice.
5. If user selects Add Expense:

   * Enter amount and category.
   * Store date automatically.
   * Save data.
6. If user selects View Expenses:

   * Display all expense records.
7. If user selects Delete Expense:

   * Select expense number.
   * Remove the selected record.
8. If user selects Show Total:

   * Calculate and display total spending.
9. If user selects Exit:

   * Save data and terminate the program.
10. End.
