#  Expense Tracker

##  Overview
Expense Tracker is a Python-based application that helps users record and manage their daily expenses. The program allows users to add, view, and delete expenses while keeping track of total spending. All data is stored in a JSON file for permanent storage.

##  Objectives
- To manage daily expenses efficiently.
- To demonstrate Python programming concepts.
- To implement file handling using JSON.
- To provide a simple and user-friendly expense management system.

##  Features
- Add new expenses
- View all recorded expenses
- Delete expenses
- Calculate total spending
- Store data permanently using JSON files

##  Technologies Used
- Python
- JSON File Handling

##  How to Run
1. Install Python.
2. Download or clone the repository.
3. Open a terminal in the project folder.
4. Run:

python expense_tracker.py
